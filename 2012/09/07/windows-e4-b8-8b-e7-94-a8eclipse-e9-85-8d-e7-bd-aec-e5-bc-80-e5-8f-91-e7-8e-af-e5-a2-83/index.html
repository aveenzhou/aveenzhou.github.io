windows下用eclipse配置c++开发环境

发表于 2012-09-07 | 分类于 C/C++ |

主要是windows下安装MinGW编译环境以及Eclispse编辑器安装CDT，C/C++ Development Tools两大步骤

通过网上收集的资料

 

将最终成功的方法总结下

一：win7下安装MinGW

1.到到http://sourceforge.net/projects/mingw/files/Installer/mingw-get-inst/上面下载最新的MinGW网上安装程序，Download mingw-get-inst-20120426.exe (662.7 kB) ，下载完成后运行此程序。

2.在安装向导中的第二步选择第二项（选第一项时始终都安不完整，不知道是否是网络的原因）

3.第三部选择所有的选项，后面就自动安装直到完成

4.设置环境变量

在path变量的之中添加C:\MinGW\bin\;C:\MinGW\msys\1.0\bin\;C:\MinGW\msys\1.0\;

5.试验下是否安装成功

进入cmd窗口，运行g++ -v,没显示错误信息就成功了

写一个简单的cpp文件

//test.cpp

#include

using namespace std;

int main()

{

cout<<”Hellow!”<<endl;

return 0;

}

切换到test.cpp文件所在的目录

执行g++ -o test.exe test.cpp回车

生成test.exe

运行test会有输出Hellow！

二、Eclispse编辑器安装CDT，C/C++ Development Tools

1.打开Eclipse，点击Help－>Install New Software
在work with后面添加网址：http://download.eclipse.org/releases/galileo，等待其更新完所有列表后，选择Programming Language下拉栏，选择Eclipse C/C++ development tools. 点击next，安装即可。


 

2.新建c++工程看是否运行成功（如果前面MinGW配置得当，这里基本上不用啥设置）
