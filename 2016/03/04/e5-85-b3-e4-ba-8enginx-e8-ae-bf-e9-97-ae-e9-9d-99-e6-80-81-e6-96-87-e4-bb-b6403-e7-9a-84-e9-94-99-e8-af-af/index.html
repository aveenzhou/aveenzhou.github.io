关于nginx访问 静态文件 403 的错误

发表于 2016-03-04 | 分类于 nginx |

例如 ngixn的配置的静态文件访问 如下：

location /static {
root /var/app/lxxxx/web;
}

1.检查所有的文件有无读权限 chmod 644 -R

2.检查父目录 即 web及以上目录有无可执行权限 chmod 755 xxx/
